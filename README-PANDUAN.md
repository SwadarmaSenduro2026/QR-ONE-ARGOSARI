# QR-One Argosari - Layout Akses Cepat

Template ini dibuat untuk versi QR-One yang lebih cepat diakses dan lebih cocok untuk scan QR dari HP.

## Fokus layout

- Menu utama ringkas.
- Tidak terlalu banyak teks di halaman awal.
- Tombol besar dan mudah ditekan.
- Cocok untuk warga, wisatawan B29, perangkat desa, dan papan informasi.
- Link dan aktivasi fitur bisa diisi nanti.

## File yang diedit

Untuk mengubah teks dan susunan menu, edit:

```text
data.js
```

Untuk mengubah warna, ukuran, dan tampilan, edit:

```text
style.css
```

## Urutan menu yang disarankan

1. Peta Desa & Lokasi Penting
2. Informasi Desa Argosari
3. Wisata B29
4. Fasilitas Umum
5. Mitigasi & Nomor Darurat
6. Kirim Data / Koreksi Informasi

## Catatan

Template ini sengaja tidak memakai folder `assets`, supaya lebih mudah di-upload ke GitHub.
Logo asli KKN/kampus bisa ditambahkan nanti setelah layout disetujui.
